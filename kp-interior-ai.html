1. Краткое резюме

Ваш проект Interior AI уже имеет работающий прототип: фронтенд на Next.js 16 + React 19, бэкенд на FastAPI с AI-анализом чертежей через GPT-4o/Claude. Основная архитектура готова.

Для перехода от прототипа к полноценному продукту необходимо реализовать 4 ключевых модуля: постоянное хранение данных, точное позиционирование элементов на плане, автоматический подбор мебели и интерактивную расстановку.

Мы предлагаем завершить разработку за 4-6 недель с возможностью поэтапной приёмки.

2. Анализ текущего состояния

Что работает

  • Загрузка и AI-анализ PDF-чертежей (определение комнат, мебели)
  • AI-парсинг каталогов мебели (до 500 страниц)
  • Интерактивный канвас с pan/zoom
  • Генерация SVG-blueprint и DALL-E рендеров
  • Переключение между AI-провайдерами (GPT-4o / Claude)

Что необходимо доработать

  • Данные хранятся в localStorage браузера (теряются при очистке кэша)
  • Позиционирование комнат на плане с погрешностью 5-15%
  • Нет автоматической связи между мебелью на плане и каталогом
  • Нет возможности перетаскивать мебель на план из каталога

7. Обоснование стоимости

Проект требует 4 области экспертизы одновременно:

  • Backend-разработка: PostgreSQL, SQLAlchemy, миграции, REST API
  • Frontend-разработка: React 19, интерактивный канвас, drag-and-drop, state management
  • Computer Vision: OpenCV edge detection, contour matching, snap-алгоритмы
  • AI-интеграция: работа с существующим AI-пайплайном (GPT-4o/Claude)

На рынке аналогичный объём работ fullstack-разработчика с экспертизой в AI и CV стоит $4,000-6,000.

Ваша экономия: один исполнитель вместо команды из 3 специалистов. Фиксированная цена без риска «растягивания» часов. Нет потерь при передаче между командами.
